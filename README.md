# Oak Angel Builders - Construction Manager

Your custom construction management app. Leads, estimates, communications — all in one place.

## Quick Deploy to Vercel (5 minutes)

### Step 1: Create GitHub Account (if you don't have one)
Go to https://github.com and sign up

### Step 2: Upload This Code to GitHub
1. Go to https://github.com/new
2. Name it: `oak-angel-app`
3. Click "Create repository"
4. On the next page, click "uploading an existing file"
5. Drag all files from this folder into the browser
6. Click "Commit changes"

### Step 3: Deploy on Vercel
1. Go to https://vercel.com and sign up with GitHub
2. Click "Add New Project"
3. Select your `oak-angel-app` repository
4. Click "Deploy" (leave all settings as default)
5. Wait 1-2 minutes
6. Your app is live at `oak-angel-app.vercel.app`!

### Step 4 (Optional): Custom Domain
1. In Vercel dashboard, go to your project
2. Click "Settings" → "Domains"
3. Add `app.oakangelbuilders.com`
4. Add the DNS records shown to your domain registrar

## Features

### Leads
- Add customer info: name, phone, email, address
- Track project type: Custom Build, Remodel, Addition, etc.
- Status pipeline: New → Contacted → Meeting → Estimate Sent → Negotiating → Won/Lost
- Communication log with dates

### Estimates
- All 11 CSI categories from your spreadsheet
- 100+ line items pre-loaded
- Auto-calculates:
  - 6.5% operating expense
  - 20% profit margin
  - Customer price
- Quick entry and full detail modes
- Real-time totals

## Data Storage
All data saves to your browser's local storage. Data persists between sessions.

For multi-device sync or team access, let me know and we can add a database.
